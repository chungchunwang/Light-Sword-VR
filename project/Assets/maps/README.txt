There are no in-built maps (for copyright reasons), so you will have to find beat saber maps on your own and place them in this folder.

You should put them in the folder specificed on the left panel of the main menu.

The main website for this Beat Saver - https://beatsaver.com

The following are the songs that I used for the demo. You can download them without an account.
